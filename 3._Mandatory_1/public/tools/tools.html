Node.js

- Node.js er et cross-platform javascript runtime miljø.
- Vi bruger det pt. sammen med express til at lave en Server.
- Man kan køre en fil i Node ved at sige node "filnavn".

Node package Manager

- Når vi downloader Node.js får vi NPM med.
- NPM - Node package manager, kan vi benytte til at downloade biblotekter som f.eks. Express og nodemon. Når vi skal benytte Express, kan vi lave en package.json fil hvor vi kan lave { }, derefter kan vi sige npm install express, hvorefter vi så får de dependencies vi har brug for.

Nodemon

- Download ved hjælp at npm: npm install -g nodemon (-g betyder at man downloader det globalt.)
- Nodemon er et super nice tool at have, da det gør man kan havde en server kørene hele tiden, så man ikke behøver at skrive node "filnavn" hver gang man vil starte ens server op.
- Gør det lidt at lave ændringer on the fly, især når vi arbejde med html og css, da man bare skal refresh ens side når man har lavet en ændring.

Cross-env

- Download ved hjælp at npm: npm install -save-dev cross-env (-save-dev, gemmer den under devDependencies)
- cross-env, er et miljø der er ens på alle maskiner, som vi så kan benytte til at lave commands, som alle maskiner ville kunne køre.
